Documentation Ecomerce API

This API provides endpoints for authentication, products, and transactions.

Endpoints

  • POST /api/auth/register - Register a new user
  • POST /api/auth/login - Login and receive a JWT
  • GET /api/productos - List all products
  • GET /api/productos/:id - Get product by ID
  • POST /api/productos - Add a new product
  • GET /api/transactions - List all transactions
  • GET /api/transactions/:id - Get transaction by ID
  • POST /api/transactions - Create a new transaction

For more details, check the API documentation in Bruno.
